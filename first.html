CSS - Cascading Style Sheets

There are 3 different ways to use CSS: Inline, Internal, External

                    We can define one style by using tag like this in the head tag.
                    <style>
                          body {background-color: powderblue;}
                          h1   {color: blue;}
                          p    {background-color: red;}
                          <style>
                

                    Other way is External links mentioning in head tag.
                    <link rel="stylesheet" href="styles.css">

                    Write a css like 
                    body {
                        background-color: powderblue;
                      }
                      h1 {
                        color: blue;
                      }
                      p {
                        color: red;
                        border: 1px solid powderblue;
                        font-family: courier;
                        font-size: 160%;
                        padding: 30px;  padding help to define space between text & border.
                      }

This is different. id tag used to define separate styling for one element alone.
